# Jumbo, a voice chat bot on Raspberry Pi

Requirements:

 * Raspberry Pi 3 board with
   * USB Sound card
   * ALSA library installed
 * Microsot Bot Builder Node.js SDK
 * Microsoft Cognitive Service & Azure subscription
 	* BING Speech API
 	* Storage Account
 	* LUIS  



To run this sample:

```javascript
cd v2
node testrpi.js
```
